# Memory

## Required Configuration

The following personal values have been removed from skill files and must be provided here before the skills can run correctly.

| Placeholder | Description | Value |
|---|---|---|
| `<github-username>` | Your GitHub username (used by the `github` collect sub-command) | _fill in_ |
| `<alphaxiv-folder-id>` | Your alphaXiv bookmarks folder UUID (used by the `alphaxiv-lib` collect sub-command) | _fill in_ |

> **How to find these values:**
> - **GitHub username**: your handle at `https://github.com/<username>`
> - **alphaXiv folder ID**: open your bookmarks folder on alphaXiv; copy the UUID from the URL `https://www.alphaxiv.org/bookmarks?folder=<uuid>`

Once filled in, skills will substitute these values wherever placeholders appear.
